# Deflationary-Token
A BEP20/ERC20 token with deflationary system. Burn + Charity + Rewards

It will charge a fee on each transaction and this is divided in:
1) Burn fee
2) Charity fee
3) Holders reward fee

You can decide you own % for each fee.
